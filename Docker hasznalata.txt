************* Docker használata *************

-Ha el akarod indítani a Docker konténert:
    docker compose up -d
    -Ha változott a Dockerfile hozzá kell adni:
        docker compose up -d --build

-Ha le akarod állítani 
    docker compose down

-Megmutatja mik futnak a stackben:
    docker compose ps

-Megmutatja az összes konténer logjait:
    docker compose logs -f
